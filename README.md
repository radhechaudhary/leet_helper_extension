# Leet Helper Extension

An AI-powered Chrome extension that enhances your LeetCode problem-solving experience by providing real-time assistance, hints, and interactive chat directly on the problem page.

## Features
1.  AI Chat Assistant
2.  Ask questions about the problem
3.  Get hints, explanations, and approaches
4.  Smart Problem Understanding
5.  Reads the current LeetCode problem automatically
6.  Provides contextual responses
7.  Overlay UI
8.  Clean floating interface on top of LeetCode
9.  No need to switch tabs
10. Code Assistance
11. Helps refine logic
12. Suggests optimizations
13.  Interactive Chat History
14. Keeps track of previous conversations
15. Interview Mode (Optional)
16. Simulates real interview environment
17. Gives minimal hints

## Tech Stack
1. Frontend: React.js
2. Extension APIs: Chrome Extension APIs
3. Backend (if used): Node.js / Express
4. AI Integration: OpenAI / Gemini API
5. State Management: React Hooks

## Installation
1. Clone the repository
    git clone https://github.com/radhechaudhary/leet_helper_extension.git
    cd leet_helper_extension
2. Install dependencies
    npm install
3. Build the extension
    npm run build
4. Load in Chrome
    Go to chrome://extensions/
    Enable Developer Mode
5. Click Load unpacked
6. Select the build or dist folder

## Usage
1. Open LeetCode
2. Go to any problem
3. Click the extension icon or open overlay
4. Start chatting with the AI assistant
